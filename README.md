# Prova_Cardes
Trabalho final de Engenharia de software
## 

História de Usuário
  HU-1:  Escolher a loja e realizar cadastro.
  •	Como um cliente.
  •	Eu quero fazer um cadastro.
  •	Para que eu possa realizar uma compra e acompanhar meus pedidos.
HU-2:  Escolher um produto.
  •	Como um cliente.
  •	Eu quero ver os produtos por nomes e categoria.
  •	Para que eu possa escolher 1, com facilidade.
HU-3:  Páginas dos produtos.
  •	Como um cliente.
  •	Eu quero ver as fotos e preços.
  •	Para que eu decido se compro.
HU-4:  Carrinho
  •	Como um cliente,
  •	Eu quero salvar e remover os produtos escolhido,
  •	Para que eu possa montar minha compra.
HU-5:  Forma de pagamentos
  •	Como um cliente,
  •	Eu quero ter opções de pagamento, pix, boleto cartões,
  •	Para que eu decido a melhor forma de pagamento.
